# Proyecto 7 - Detector de Idioma

## Descripción

Aplicación desarrollada en Python con Tkinter que permite detectar automáticamente el idioma de un texto introducido por el usuario.

## Características

- Detección automática del idioma.
- Muestra el nombre del idioma.
- Muestra el código ISO del idioma.
- Indica el porcentaje aproximado de confianza.
- Validación de textos demasiado cortos.
- Interfaz gráfica sencilla.

## Tecnologías utilizadas

- Python 3
- Tkinter
- langdetect
