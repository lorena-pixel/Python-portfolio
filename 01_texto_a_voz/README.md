# Proyecto 1 - Texto a Voz

## Descripción

Este programa permite convertir el texto de un artículo web en un archivo de audio MP3.

El usuario introduce la URL de un artículo y el programa:

- Descarga el contenido de la página.
- Extrae el texto principal.
- Convierte el texto en voz.
- Genera un archivo llamado `articulo.mp3`.

## Tecnologías utilizadas

- Python 3
- newspaper3k
- gTTS

## Instalación

Instalar las dependencias:

```bash
pip install -r requirements.txt
```

## Ejecución

Ejecutar el programa:

```bash
python main.py
```

Después introduce la URL del artículo que deseas convertir.

## Resultado

El programa genera un archivo de audio:

```
articulo.mp3
```
