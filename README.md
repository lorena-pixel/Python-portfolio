# 📖 Descripción de los proyectos

## 🎤 01 - Texto a Voz

Aplicación que descarga el contenido de un artículo web y lo convierte automáticamente en un archivo de audio MP3.

**Tecnologías**

- newspaper3k
- gTTS

---

## 🏓 02 - Pong

Recreación del clásico Pong.

Incluye:

- Movimiento de jugadores
- Rebote de pelota
- Sistema de puntuación
- Detección de colisiones

---

## ⌨️ 03 - Escritura Veloz

Aplicación para practicar mecanografía y medir la velocidad de escritura.

---

## 📝 04 - Editor de Texto

Editor de texto desarrollado con Tkinter.

Permite:

- Crear documentos
- Abrir archivos
- Guardar archivos

---

## 🧩 05 - Sudoku

Resolutor automático de Sudoku mediante Backtracking.

Incluye:

- Versión de consola
- Versión con interfaz gráfica

---

## 🌐 06 - Comprobador Web

Comprueba el estado de una página web mediante peticiones HTTP.

---

## 🌍 07 - Detector de Idioma

Detecta automáticamente el idioma de un texto utilizando procesamiento estadístico del lenguaje.

---

## ⏰ 08 - Reloj Despertador

Aplicación de escritorio que muestra un reloj en tiempo real y permite configurar alarmas que abren automáticamente un vídeo de YouTube.

---

## 🎬 09 - Recomendador Netflix

Sistema de recomendación basado en similitud utilizando un conjunto de datos de Netflix.

Características actuales:

- Versión consola
- Versión gráfica
- Búsqueda por título
- Recomendaciones similares
- Información detallada del contenido
