# Proyecto 6 - Comprobador de Conectividad Web

## Descripción

Aplicación desarrollada con Python y Tkinter que permite comprobar si un sitio web está disponible.

El usuario introduce una dirección web y el programa verifica la conexión mostrando el código de respuesta HTTP.

## Características

- Introducción de cualquier URL.
- Comprobación automática del sitio.
- Detección de errores de conexión.
- Visualización del código HTTP.
- Interfaz gráfica sencilla.

## Tecnologías utilizadas

- Python 3
- Tkinter
- urllib
