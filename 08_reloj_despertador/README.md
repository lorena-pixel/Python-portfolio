# Proyecto 8 - Reloj Despertador

## Descripción

Aplicación desarrollada en Python con Tkinter que permite configurar una alarma para una hora determinada.

Cuando llega la hora indicada, el programa abre automáticamente un vídeo aleatorio de YouTube almacenado en un archivo de enlaces.

## Características

- Reloj digital en tiempo real.
- Configuración de alarma.
- Activar y desactivar alarma.
- Apertura automática de un vídeo de YouTube.
- Selección aleatoria del vídeo.

## Tecnologías utilizadas

- Python 3
- Tkinter
- datetime
- webbrowser
