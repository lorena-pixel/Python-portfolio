# Proyecto 4 - Editor de Texto

## Descripción

Aplicación desarrollada en Python con Tkinter que simula un editor de texto sencillo.

El programa permite crear, abrir, editar y guardar archivos de texto mediante una interfaz gráfica.

## Características

- Crear un documento nuevo.
- Abrir archivos de texto (.txt).
- Editar el contenido.
- Guardar cambios.
- Guardar como un nuevo archivo.
- Interfaz gráfica intuitiva.

## Tecnologías utilizadas

- Python 3
- Tkinter
