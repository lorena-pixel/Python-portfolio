# Proyecto 3 - Prueba de Escritura Veloz

## Descripción

Aplicación desarrollada con Tkinter que permite medir la velocidad y precisión de escritura del usuario.

## Características

- Frases aleatorias.
- Medición del tiempo.
- Cálculo de palabras por minuto (PPM).
- Cálculo del porcentaje de precisión.
- Interfaz gráfica sencilla e intuitiva.

## Tecnologías utilizadas

- Python 3
- Tkinter

## Ejecución

```bash
python main.py
```
