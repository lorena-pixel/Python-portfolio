# Proyecto 2 - Pong

## Descripción

Recreación del clásico juego Pong utilizando el módulo `turtle` de Python.

## Características

- Dos jugadores.
- Marcador automático.
- Rebote de la pelota.
- Reinicio de la partida tras cada punto.
- Aumento progresivo de la velocidad de la pelota.

## Controles

### Jugador izquierdo

- W → Subir
- S → Bajar

### Jugador derecho

- ↑ Flecha arriba → Subir
- ↓ Flecha abajo → Bajar

## Tecnologías utilizadas

- Python 3
- Turtle

## Ejecución

```bash
python main.py
```
